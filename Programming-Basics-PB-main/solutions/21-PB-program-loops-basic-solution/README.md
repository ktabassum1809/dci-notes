# Loops Basic

## Task 01

Fix the code below so that it prints the numbers from 0 to 10

```javascript
for (let i = 0; i < 10; i--) {
    console.log(i);
}
```

## Task 02

Complete the code below so that it prints the numbers from 100 to 50



## Task 03

Complete the code below so that output will be a string with 5 +

Expected output: 
```plaintext
+++++
```


```javascript
let output = "";

for () {
  output += "*";
}

console.log(output);
```
